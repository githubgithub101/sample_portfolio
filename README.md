# Juan Dela Cruz - Sample Portfolio Website

This is a sample portfolio website created for Module 6 learning task demonstration.

## Project Structure

```
sample_web/
│
├── index.html          # Home page
├── about.html          # About page
├── projects.html       # Projects page
├── contact.html        # Contact page
│
├── css/
│   └── styles.css      # Main stylesheet
│
├── images/
│   ├── profile.jpg     # Profile picture (SVG)
│   ├── project1.jpg    # Project 1 image (SVG)
│   ├── project2.jpg    # Project 2 image (SVG)
│   └── project3.jpg    # Project 3 image (SVG)
│
├── how_to_publish.md   # GitHub Pages publishing guide
└── README.md           # This file
```

## Features

### ✅ All Required Pages
- **Home**: Welcome message, introduction, call-to-action buttons
- **About**: Personal information, education, skills (6), interests (4)
- **Projects**: 3 complete projects with descriptions and technologies
- **Contact**: Contact information, social links (4), contact form

### ✅ Design Features
- Professional color scheme (Purple gradient theme)
- Responsive navigation menu with hover effects
- Active page highlighting
- Consistent styling across all pages
- Mobile-responsive design with @media queries
- Professional typography and spacing
- Interactive elements (hover effects, transitions)

### ✅ Technical Requirements
- Valid HTML5 structure
- External CSS stylesheet
- Semantic HTML tags
- Proper folder organization
- Responsive design
- All links functional
- Images with alt text

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Styling and layout
- **SVG**: Placeholder images (scalable vector graphics)

## How to View Locally

1. Navigate to the `sample_web` folder
2. Double-click `index.html`
3. The website will open in your default browser
4. Test all navigation links and features

## How to Publish to GitHub Pages

See the comprehensive guide in **`how_to_publish.md`** for step-by-step instructions on:
- Creating a GitHub account
- Setting up Git
- Uploading your files
- Enabling GitHub Pages
- Accessing your live website

## Customization Tips

Students can customize this template by:
1. Replacing placeholder images with real photos
2. Updating personal information in all pages
3. Changing the color scheme in `styles.css`
4. Adding more projects to the Projects page
5. Modifying the contact form fields
6. Adding new sections to any page

## File Naming Conventions

- All file names use lowercase
- HTML files: `index.html`, `about.html`, etc.
- Folders: `css`, `images`
- No spaces in file names (use hyphens if needed)
